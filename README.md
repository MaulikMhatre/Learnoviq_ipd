# EduRath AI (Learnoviq) 🚀

**EduRath AI** (also known as Learnoviq) is an intelligent, adaptive learning gateway designed specifically for students in Tier 2 and Tier 3 cities across India. By aggregating elite academic content from platforms like **NPTEL, Swayam, and Coursera**, EduRath AI ensures that high-quality education is accessible regardless of local bandwidth constraints or prior academic background.

---

## 🌟 Key Features

- **📡 Adaptive Bandwidth Optimization**: Automatically adjusts content delivery (high-compression video, text-first nodes) based on real-time network detection.
- **🗺️ AI-Driven Mastery Paths**: Generates personalized, node-based learning roadmaps that adapt to your speed and proficiency.
- **🛡️ Digital Skill Passport**: A verified competency portfolio recognized by industry partners, synced with blockchain for immutable certification.
- **🏆 Sadhana Network**: A district-level leaderboard and social learning environment that fosters healthy competition and collaboration.
- **🤖 Intelligence Engine**: Continuously analyzes performance to bypass mastered concepts and flag missing prerequisites.

---

## 📂 Page Breakdown

### 1. 🏠 Landing Page (`/`)
The entry point of the platform, designed with a professional "Digital India" aesthetic.
- **Hero Section**: High-impact messaging about digital mastery for Indian youth.
- **Value Proposition**: Highlights optimized connectivity, passport verification, and the adaptive engine.
- **Social Proof**: Showcases integration with national platforms like NPTEL and SWAYAM.

### 2. 📊 Student Dashboard (`/dashboard`)
A personalized command center for the learner.
- **Proficiency Tracking**: Real-time stats on nodes mastered and regional ranking (e.g., "Top 2% of Pune District").
- **Active Path Summary**: A quick view of the current "Mastery Path" (e.g., AI Architect) with a progress timeline.
- **Sadhana Leaderboard**: Localized rankings to motivate learners within their own districts.
- **Regional Events**: Announcements for local hackathons and state-level challenges.

### 3. 🔍 Course Discovery (`/discovery`)
A unified catalog aggregating the best of Indian and global academia.
- **Institutional Filters**: Quickly toggle between NPTEL, Swayam, Coursera, and EduRath's own local modules.
- **Adaptive Filtering**: A unique feature that highlights courses compatible with the user's current bandwidth profile.
- **Academic Hierarchy**: Filter by Undergraduate, Postgraduate, or Professional Certification levels.
- **Rich Course Cards**: Displays student counts, ratings, and institutional badges.

### 4. 🛂 Digital Skill Passport (`/passport`)
A comprehensive, verified professional identity.
- **Cognitive Mastery Radar**: A visual breakdown of proficiency across domains (Python, Frontend, ML, etc.).
- **Verified Credentials**: A list of SHA-256 verified certificates from partner institutions.
- **Platform Sync**: Integrates learner IDs from GitHub, NPTEL, and IITs into a single verified profile.
- **Adaptive Recommendations**: AI-driven suggestions for the next credential needed to reach a career goal.

### 5. 🛤️ Mastery Path (`/path`)
An interactive, node-based knowledge graph.
- **Interactive Graph**: Visualizes the learning journey with core and elective nodes (Completed, Active, Locked).
- **AI Optimization**: Users can "Optimize Path" to let the AI re-route their journey based on recent performance scores.
- **Academic Advisory**: Provides warnings for missing prerequisites and recommends remedial paths to ensure high retention.
- **Module Details**: Deep-dive into the active node with time estimates and integration details.

## 🛠️ Technology Stack

- **Frontend**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **State Management**: Zustand (implied by `src/store`)
- **Typography**: Display & Sans fonts optimized for readability

## 🚀 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
4.  **Open in browser**: Navigate to `http://localhost:3000`
